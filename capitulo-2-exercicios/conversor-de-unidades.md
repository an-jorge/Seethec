# Conversor de Unidades

{% hint style="warning" %}
### ⚡️ Exercício 5: Conversor de Unidades <a href="#exercicio-1-operacoes-matematicas-simples" id="exercicio-1-operacoes-matematicas-simples"></a>

**Objetivo:** Criar um programa que converta uma medida de **quilômetros para outra unidade** (metros, centímetros, milímetros ou milhas).\

{% endhint %}

### **Instruções:** <a href="#instrucoes" id="instrucoes"></a>

* [ ] Peça ao usuário para digitar um valor em quilômetros.
* [ ] Peça a unidade de conversão desejada:
  * `1` para Metros
  * `2` para Centímetros
  * `3` para Milímetros
  * `4` para Milhas
* [ ] Use `switch` para converter corretamente.

**Exemplo de entrada e saida**

{% code overflow="wrap" %}
```

Digite o valor em quilômetros: 5

Escolha a conversão
1-Metros, 
2-Centímetros,
3-Milímetros, 
4-Milhas
 
 >>>: 2

Resultado: 5.00 km equivalem a 500000.00 cm.

```
{% endcode %}



***



{% hint style="info" %}
#### **Conversão de unidades na Física**



* 1 km = **1.000 metros**
* 1 km = **100.000 centímetros**
* 1 km = **1.000.000 milímetros**
* 1 km = **0.621371 milhas**


{% endhint %}



***



### **Desafios Extras:**

#### **Desafios Extras:**

* Permita que o usuário faça várias conversões sem reiniciar o programa.
* Se o usuário digitar um número inválido, peça para ele tentar novamente.
