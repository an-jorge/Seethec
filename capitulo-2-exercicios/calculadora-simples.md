# Calculadora Simples

{% hint style="warning" %}
### ⚡️ Exercício 4: Calculadora Simples <a href="#exercicio-1-operacoes-matematicas-simples" id="exercicio-1-operacoes-matematicas-simples"></a>

**Objetivo:** Criar um programa que peça ao usuário dois números e uma operação matemática

&#x20;(`+`, `-`, `*`, `/`) e exiba o resultado.\

{% endhint %}

### **Instruções:** <a href="#instrucoes" id="instrucoes"></a>

* [ ] Peça ao usuário para digitar dois números.
* [ ] Peça o operador (`+`, `-`, `*`, `/`).
* [ ] Use `switch` para executar a operação correta.
* [ ] Se o usuário tentar dividir por zero, exiba um aviso.



**Exemplo de entrada e saida**

{% code overflow="wrap" %}
```
Digite dois números: 10 5
Escolha a operação (+, -, *, /): *
Resultado: 5 * 5 = 25

```
{% endcode %}



***



### **Desafios Extras:**

* Permita que o usuário realize mais de uma operação sem reiniciar o programa.
* Exiba uma mensagem personalizada se o usuário tentar dividir por zero.
